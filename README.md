# Sistema de Inventario POO

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un sistema de gestión de inventario utilizando el lenguaje Java y aplicando los principios de Programación Orientada a Objetos.

El sistema permite registrar, consultar, buscar y eliminar productos dentro de un inventario mediante una aplicación de consola.

## Tecnologías utilizadas
- Java
- NetBeans IDE
- Programación Orientada a Objetos

## Conceptos de POO aplicados
En el desarrollo del proyecto se aplicaron los siguientes conceptos:

- Herencia
- Polimorfismo
- Abstracción
- Uso de colecciones (ArrayList)
- Manejo de excepciones

## Estructura del proyecto

El sistema está compuesto por las siguientes clases:

- **Producto.java**  
Clase base que representa las características generales de un producto.

- **ProductoElectronico.java**  
Clase que hereda de Producto y representa productos electrónicos.

- **ProductoAlimento.java**  
Clase que hereda de Producto y representa productos alimenticios.

- **Inventario.java**  
Clase encargada de gestionar la lista de productos del inventario.

- **InventarioPOO.java**  
Clase principal que contiene el método `main` y permite ejecutar el sistema.

## Funcionalidades del sistema

El sistema permite realizar las siguientes operaciones:

1. Agregar producto electrónico
2. Agregar alimento
3. Mostrar productos
4. Buscar producto
5. Eliminar producto
6. Salir del sistema
